<h1>Conversor de Moedas</h1>
<p>Este é meu primeiro projeto em Java. Ele utiliza da API ExchangeRate para fazer a conversão entre as moedas selecionadas, dentro do escopo de moedas escolhido.</p>
<h2>Passo a passo:</h2>
<ul>
  <li>Selecione a conversão que deseja realizar com base na lista apresentada no console.</li>
  <li>Digite o valor que deseja converter, como inteiro ou decimal.</li>
  <li>Você receberá a resposta da moeda de origem e seu valor convertido para a moeda destino e seu valor.</li>
</ul>
